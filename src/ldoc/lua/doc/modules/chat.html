Module chat

Functions for dealing with chat messages.

Any chat channels or message receivers are indexed to this module, allowing for methods.

Functions

send (target, message[, type='system']) Sends a message to a chat channel or message receiver.
send_from (target, sender, message[, type='system']) Sends a message from a sender to a chat channel or message receiver.
fixed (arg) Creates a chat channel from some number of message receivers.
combined (arg) Creates a chat channel from some number of other channels.
permission (permission) Creates a chat channel for anyone with a given permission.
world (world) Creates a chat channel for anyone in a given world.

Tables
